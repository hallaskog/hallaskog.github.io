Portracer

Portracer is a lan-multiplayer racing game.

You can connect up to 8 Clients via IP/Port.

The race is won when one of the players crosses the finish line.

The game is written in Java and is run over a .jar file.

If you can't run the game by double clicking on the jar

open up a cmd window and navigate to the dictionary

where you saved the file.

Then run the command 'java -jar portracer.jar'.


Have fun!
